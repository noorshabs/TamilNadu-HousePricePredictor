# 🏠 Tamil Nadu House Price Predictor

A Machine Learning web application that predicts house prices in Chennai based on property details using a Random Forest Regression model.

---

## 📌 Project Overview

This project predicts the estimated selling price of houses in Chennai using Machine Learning. Users can enter property details such as area, number of bedrooms, location, and amenities through an interactive Streamlit web application.

The prediction is generated using a trained Random Forest Regression model.

---

## ✨ Features

- 🏡 Predict house prices instantly
- 📍 Choose from multiple Chennai locations
- 🛏 Select number of bedrooms
- 📐 Enter property area
- ✅ Select amenities using checkboxes
- ⚡ Fast predictions using Random Forest
- 🎨 Interactive Streamlit interface

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/noorshabs/TamilNadu-HousePricePredictor.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
